# ipsec-app
IPsec project

# How to install for devs
## 1. npm
$ npm install
## 2. python virtual env
$ python3 -m venv venv<br>
## 3. activate venv
(windows) .\env\Scripts\activate<br>
(linux) source env/bin/activate<br>
## 4. install python libs
$ pip install -r requirements.txt

# How to run
npm start
